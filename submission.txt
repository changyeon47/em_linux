개발 환경
------------------------------------------------------------
OS:             Ubuntu Linux (x86_64)
Shell:          bash

Backend:        PHP 8.x + MySQLi
Frontend:       Bootstrap 5, 순수 JavaScript

파일 구조
------------------------------------------------------------
em_linux/
├── db.php          # MySQL 접속 설정 (DB_HOST, DB_USER, DB_PASS, DB_NAME)
├── index.php       # 실시간 웹 대시보드 (Bootstrap 5 다크 테마, 3초 갱신)
├── api.php         # JSON REST API (위치별 최신 데이터 + 최근 20개 기록)
├── injector.py     # Python 센서 데이터 시뮬레이터 (3초 주기 INSERT)
├── setup.sh        # 자동 설치 스크립트 (DB + Apache + 권한 설정)
├── README.md       # 사용 설명서 (한국어)
├── process.md      # 프로젝트 문서 (Mermaid 플로우차트 포함)
└── submission.txt  # 이 파일
